# PLUGINS

* This directory contains Javascript plugins that you want to run before mounting the root Vue.js application.

## Usage
* API : `api.js`

* SEO : `seo.js`
  * General : `this.$generalSeo()`

* Icons : `icons.js`
  * `<v-icon name="" />`
